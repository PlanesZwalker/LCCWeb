🎮 Letters Cascade

Bienvenue dans le jeu de lettres en cascade !
Formez des mots en plaçant les lettres qui tombent et obtenez le meilleur score.

🎯 Comment jouer :

  • Utilisez les flèches ← → pour déplacer les lettres
  • Appuyez sur Entrée pour placer la lettre
  • Formez des mots pour gagner des points
  • Évitez que la grille se remplisse complètement
  • Plus vous formez de mots, plus le niveau augmente
